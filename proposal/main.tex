\chapter{Objetivos}\label{cap:objetivos}

O principal objetivo deste trabalho consiste em projetar, implementar e avaliar uma linguagem de programação similar a linguagem de programação c.
Esta implementação de linguagem de programação devera fazer uso do ecosistema llvm, incluindo suas ferramentas para geração de linguagem de representação intermediaria, para otimização de código e para geração de codigo objeto executavel.
